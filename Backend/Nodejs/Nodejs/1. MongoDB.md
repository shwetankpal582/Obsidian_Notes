		In SQL:
			SELECT * FROM users WHERE age > 21;		
		In MongoDB:
			db.users.find({find {condition}})
				db.users.find({ age: { $gt: 21 }});

5. Update Data:
		In SQL:
			UPDATE users SET age =22 WHERE username = 'Alice';
		In MongoDB:
			db. users. updateOne({ username: 'Alice' } , { $set: { age: 22}});
6. Delete Data:
