# Linked List Inserting At End
Given the head of a Singly Linked List and a value x, insert that value x at the end of the LinkedList and return the modified Linked List

> [!Example] Example
> **INPUT:**  LinkedList: 1-> 2-> 3-> 4-> 5, x = 6
> **OUTPUT:** 1->2->3->4->5->6
